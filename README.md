# 🇮🇳 Praja - AI Voice FIR Assistant

An AI-powered voice assistant for filing FIR (First Information Report) in India. Built with Sarvam AI for superior Indian language support.

## 🚀 Quick Start

### Run Server
```bash
cd backend
python server.py
```

Open: **https://localhost:5000**

## 📁 Project Structure

```
Praja/
├── frontend/       # Frontend (HTML, JS, CSS)
├── backend/        # Flask API server
├── ai/sarvam/      # Sarvam AI integration
├── config/         # .env, certificates
└── docs/           # Documentation
```

See full structure in project README files.

## ✨ Features

- 🎤 Voice input in Telugu, Hindi, English
- 🤖 AI-powered conversation (Sarvam AI)
- 🔊 Natural Indian voice responses
- ⏱️ Smart 4-second pause detection
- 🔄 Continuous conversation mode
